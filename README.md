# Coach Sync

1-on-1 coaching calls between students and coaches.
This application manages their scheduling 


## Technical Stack and Setup:

Ruby version: 3.4.1

Rails version: 8.0.2


System dependencies
Use Tailwind CSS for UI 



Database 
- Postgres DB

### Database initialization 

`bin/rails db:create`

`bin/rails db:migrate`

Clears the database and adds initial data
`bin/rails db:seed`

Once you have the  app installed 
`cd coschsync`

Run `bundle install` for app dependencies
    


#### How to run the test suite

`rspec` - to run the entire suite
